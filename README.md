**Тема:** Разработка программного модуля для учета заявок на ремонт климатического оборудования (Вариант 5)

## Описание проекта
Разработка веб-приложения для автоматизации учёта заявок на ремонт климатического оборудования (кондиционеры, вентиляция, увлажнители и т.д.). Система позволяет регистрировать заявки, назначать ответственных, отслеживать статусы, вести статистику, а также добавлять роли и дополнительные функции согласно техническому заданию демонстрационного экзамена.

## Основные функции модуля (согласно ТЗ)
- Добавление, редактирование и просмотр заявок на ремонт.
- Регистрация заявок оператором.
- Обработка и назначение исполнителя (специалиста).
- Исполнение заявки с возможностью добавления комментариев и информации о запчастях.
- Отчётность: количество выполненных заявок, среднее время ремонта, статистика по типам неисправностей.
- Мониторинг и анализ эффективности работы сервисной службы.
- **Дополнительно (по Прил_3):**
  - Роль «Менеджер по качеству» с правами: привлечение специалистов, продление срока заявки.
  - Генерация QR-кода со ссылкой на форму обратной связи для оценки качества обслуживания.

## Стек технологий

### Backend
- C#
- ASP.NET Core 8 (Web API)
- Entity Framework Core (ORM)
- PostgreSQL 17
- Инструменты: DBeaver, pgAdmin 4

### Frontend
- HTML5, CSS3
- JavaScript (нативный, без фреймворков)
- Адаптивный интерфейс с модальными окнами
